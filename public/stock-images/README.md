## Image Specifications

- **Format**: JPG or PNG
- **Recommended Resolution**: 
  - Hero images: 1600x900px or higher
  - Card images: 1200x800px or higher
  - Aspect ratios should be maintained for responsive layouts
- **Optimization**: Images should be optimized for web to ensure fast loading times
- **Content**: All images should feature Indian professionals, students, or settings as appropriate

All images are configured with responsive classes:
- `object-cover` for proper cropping

- `loading="lazy"` for performance optimization
- Proper alt text for accessibility
